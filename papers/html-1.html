Section A: Multiple Choice Questions (1 mark each)

Attempt all questions (10 Marks)

1. What does HTML stand for?
a) HyperText Marking Language
b) HyperText Markdown Language
c) HyperText Markup Language
d) None of the above
2. Which HTML tag is used to define an internal style sheet?
a) <style>
b) <css>
c) <script>
d) <link>
3. Which tag is used to create a hyperlink in HTML?
a) <link>
b) <a>
c) <href>
d) <hyperlink>
4. What is the correct HTML element for inserting a line break?
a) <br>
b) <lb>
c) <break>
d) <line>
5. Which attribute is used to provide a unique name to an HTML element?
a) class
b) id
c) name
d) tag
6. What is the correct way to add an image in HTML?
a) <img href="image.jpg">
b) <image src="image.jpg">
c) <img src="image.jpg" alt="Description">
d) <src img="image.jpg">
7. How can you make a numbered list in HTML?
a) <list>
b) <ol>
c) <ul>
d) <dl>
8. Which tag is used to define a table row in HTML?
a) <td>
b) <th>
c) <tr>
d) <table>
9. What does the <head> element contain?
a) Content shown in the browser
b) Metadata about the document
c) Page background
d) User information
10. Which tag is used to define an unordered list?
a) <ol>
b) <ul>
c) <li>
d) <list>
